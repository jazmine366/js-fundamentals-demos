# JavaScript Fundamentals

A collection of beginner JavaScript projects built while learning core JavaScript concepts such as variables, functions, arrays, objects, DOM manipulation, events, and basic application state management.

## Projects

### 1. Counter App

A simple counter that allows users to:

- Increase count
- Decrease count
- Reset count

Concepts used:

- Variables (`let`, `const`)
- Functions
- DOM manipulation
- Event listeners
- `textContent`

---

### 2. To-Do List

A task management app that allows users to:

- Add tasks
- Mark tasks as completed
- Delete tasks
- Restart task tracking

Concepts used:

- Arrays
- User input
- Dynamic HTML creation
- `createElement()`
- `appendChild()`
- Event listeners
- `filter()`
- DOM manipulation

---

### 3. Quiz App

An interactive multiple choice quiz that allows users to:

- Answer questions
- Track score
- Move through multiple questions
- Restart the quiz

Concepts used:

- Objects
- Arrays of objects
- Loops (`forEach`)
- Dynamic button creation
- Conditional logic
- Application state management
- DOM manipulation
